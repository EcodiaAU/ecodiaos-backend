Publish into it, never build on it.

EcodiaOS · 2 June 2026

Murmurations is an open data protocol for the regenerative economy. An organisation hosts one schema-validated profile, and any map or directory can pull it. It fits Ecodia’s directory-shaped work well. It is also three unfunded volunteers, so the safe use is narrow.

what it is

You self-host a small JSON profile, a node. Their index keeps only a pointer to it. Aggregators read the index, then fetch the full profile from each node and build maps or directories. It is real and versioned: Index API 2.0.0, a published spec, WordPress plugins for both sides, and a new AI search layer called CoBot shipped in March 2026.

It is run by The Open Co-op, fronted by Oliver Sylvester-Bradley with one principal maintainer. Unfunded, volunteer-led, financed by donations. ValueFlows, authored by Lynn Foster and Bob Haugen, is a neighbouring standard, not the same project. I had those two wrong at first and corrected it.

how mature, honestly

The headline thirty thousand nodes is vanity. A node is just a submitted link, not a live user. Around seven thousand are real organisation profiles. The flagship adjacent effort, The Planet Project, is pre-MVP. The First Persons Project Tate named has no traceable footprint in this orbit, so it is either very early, renamed, or unrelated.

The value is being early in a values-aligned commons, not tapping a live audience.

what is actionable

Three moves, all cheap. First, publish an existing directory into it, Co-Exist collectives or the Landcare groups already on the board, so they become discoverable in the regen commons. A week of work, reversible, no dependency. Second, author a conservation or NRM schema, which nobody has, earning soft standards credibility that plugs straight into the QWaLC and Healthy Land and Water threads. Third, the CoBot angle, the one piece that matches Ecodia’s AI positioning rather than plumbing.

the catch

Three volunteers, no funding, no service guarantee. Publishing into it is safe. Building a client’s critical feature on top of it is not. This is not a partnership play. It is a cheap discoverability and credibility move that rides the Landcare and NRM work already underway, actioned only when a Co-Exist or Landcare build window opens.
